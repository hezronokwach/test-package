## [1.0.0] - 2025-01-15

### Added
- Initial release of Unixar Mappers package
- Core mapper function with 18+ model type support
- TypeScript type definitions for all models
- Support for the following mappers:
  - `academic_levels`
  - `academic_terms`
  - `academic_titles`
  - `academic_qualifications`
  - `academic_years`
  - `campuses`
  - `cohorts`
  - `countries`
  - `course_units`
  - `courses`
  - `departments`
  - `digital_file_types`
  - `genders`
  - `honorifics`
  - `institutions`
  - `intakes`
  - `marital_statuses`
  - `religions`
  - `semesters`
- Comprehensive unit tests
- Full TypeScript support with type exports
- GitHub installation support
- Documentation and examples
